# 📊 Smart Student Productivity Tracker

## 📌 Introduction
Students often struggle with managing their study time effectively. This project provides a simple tool to track, analyze, and improve study habits using data.

---

## ❗ Problem Statement
Many students do not have a structured way to monitor their study hours, leading to poor time management and inconsistent productivity.

---

## 🎯 Objectives
- Track daily study hours
- Analyze subject-wise effort
- Visualize productivity trends
- Provide performance feedback

---

## 🚀 Features
- Add study data (subject + hours)
- Calculate total study time
- Save data to CSV file
- Generate bar graph visualization
- Predict performance level

---

## 🛠️ Tech Stack
- Python
- Matplotlib
- CSV (for data storage)

---

## ⚙️ How It Works
1. User enters study data
2. Data is stored in memory
3. User can save it as CSV
4. System analyzes total hours
5. Graphs are generated for visualization
6. Performance prediction is shown

---

## ▶️ How to Run

```bash
pip install matplotlib
